## Base URL

```
Production:  https://api.cryptoserve.io/v1
Development: http://localhost:8001
```

## Authentication

All API requests require a JWT token:

```bash
curl -H "Authorization: Bearer <token>" \
  http://localhost:8001/api/crypto/encrypt
```

## Response Format

### Success

```json
{
  "data": { ... },
  "warnings": []
}
```

### Error

```json
{
  "error": "error_code",
  "message": "Human-readable description",
  "details": { ... }
}
```

## Status Codes

| Code | Description |
|------|-------------|
| `200` | Success |
| `201` | Created |
| `400` | Bad Request |
| `401` | Unauthorized |
| `403` | Forbidden |
| `404` | Not Found |
| `429` | Rate Limited |
| `500` | Server Error |
