## Prerequisites

- **Docker** and Docker Compose
- **Python 3.11+** for the SDK
- **GitHub account** for OAuth authentication

## Architecture Overview

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│                 │     │                 │     │                 │
│   Your App      │────▶│   CryptoServe   │────▶│   Database      │
│   + SDK         │     │   Backend       │     │   + Keys        │
│                 │     │                 │     │                 │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                       │
        │                       ▼
        │               ┌─────────────────┐
        │               │                 │
        └──────────────▶│   Dashboard     │
                        │                 │
                        └─────────────────┘
```

Your application uses the SDK to communicate with the CryptoServe backend. The dashboard provides a UI for managing identities, contexts, and viewing audit logs.
